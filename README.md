# LLM-powered-Q-A-system-using-LangChain-and-Groq
Updated and modernized a 3-year-old LangChain project from Kaggle for compatibility with the latest LangChain ecosystem. This project demonstrates how to build an LLM-powered Q&amp;A system using LangChain and Groq that transcribes YouTube videos and answers user questions based on the video content. Original concept credits to Derrick Mwiti on Kaggle.
